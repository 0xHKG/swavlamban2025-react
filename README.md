# Swavlamban 2025 - React Frontend

Modern, responsive web application for the Indian Navy's Swavlamban 2025 event registration system.

## Features

- ✅ **Modern UI/UX** - Material-UI components with Indian Navy theme
- ✅ **Fast & Responsive** - Built with Vite + React + TypeScript
- ✅ **Instant Navigation** - Client-side routing (no page reloads!)
- ✅ **Role-Based Access** - Admin and User roles
- ✅ **Real-time Data** - Live dashboard statistics
- ✅ **Mobile Friendly** - Fully responsive design

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **UI Library**: Material-UI (MUI)
- **Routing**: React Router v6
- **Data Grid**: MUI X Data Grid
- **Styling**: Emotion (CSS-in-JS)

## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

App will be available at `http://localhost:5173`

### Default Login
```
Username: admin
Password: admin123
```
